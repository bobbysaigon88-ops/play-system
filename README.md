# Play System

Generative layout engine for editorial content.

## Concept

This system separates **intent from execution**:

- The author defines **importance** (Card Size)
- The system determines **composition** (layout, rhythm, contrast)

The result is a living index where layout is not designed — it is **interpreted**.

---

## V1 — Foundations

- Continuous board (no fixed sections)
- CMS-driven card sizing (2C / 3C / 4C / 6C)
- Aspect ratio selection with contrast rules
- Y-axis choreography (high / mid / low bands)
- Collision avoidance (no overlap)

---

## Philosophy

Design is no longer about placing elements.

It is about defining:
- constraints
- rules
- behavior

The system owns the aesthetic.
The human owns the intent.

---

## Stack

- Webflow (CMS + structure)
- JavaScript (layout engine)
